# 🔗 Cybersecurity Resources

## 📚 General Resources

* [PayloadsAllTheThings](https://github.com/swisskyrepo/PayloadsAllTheThings/tree/master) - Large collection of payloads, techniques, and security references.
* [WaybackURLs](https://github.com/tomnomnom/waybackurls) - Fetch historical URLs from the Wayback Machine.
* [Internet Archive Wayback Machine](https://archive.org/web/) - Browse historical versions of websites.

---

## 🔍 Reconnaissance & Intelligence

* [Shodan](https://www.shodan.io/) - Search engine for internet-connected devices and exposed services.
* [Exploit Database](https://www.exploit-db.com/) - Public exploits, CVEs, and proof-of-concept code.

---

## 📖 Reference & Cheat Sheets

* [PentestMonkey](https://pentestmonkey.net/) - Reverse shells, SQL injection, privilege escalation, and general pentesting cheat sheets.
* [File Signatures (Magic Bytes)](https://en.wikipedia.org/wiki/List_of_file_signatures) - Common file headers and hex signatures.

---

## ⬆️ Privilege Escalation

* [GTFOBins](https://gtfobins.org/) - Unix binaries that can be abused for privilege escalation.
* **pspy64** - Process monitoring tool for Linux privilege escalation.

```bash
wget https://github.com/DominicBreuker/pspy/releases/latest/download/pspy64
```

---

## 🔐 Encoding, Decoding & Hashes

* [CyberChef](https://gchq.github.io/CyberChef/) - Decode, encode, analyze, and transform data.
* [JWT.io](https://www.jwt.io/) - Decode and inspect JSON Web Tokens.
* [QuipQiup](https://quipqiup.com/) - Cryptogram and substitution cipher solver.
* [Hashes.com](https://hashes.com/en/decrypt/hash) - Lookup known hash plaintexts.
* [CrackStation](https://crackstation.net/) - Recover plaintext passwords using large precomputed databases.
